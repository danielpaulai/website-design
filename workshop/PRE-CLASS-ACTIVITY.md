# Pre-Class Activity

Before tomorrow's workshop, set up your laptop using the platform-specific guide below. Total time: ~25 minutes.

---

## Choose your guide

### 🍎 On a Mac? → **[MAC-GUIDE.md](MAC-GUIDE.md)**

### 🪟 On Windows? → **[WINDOWS-GUIDE.md](WINDOWS-GUIDE.md)**

Each guide walks you through the same five things, with the right commands and screenshots for your platform:

1. **Install Claude Code** (the command-line tool — not the consumer chat app)
2. **Install both workshop skills** in one paste — Seedance Loop Prompt Builder + Frontend Design
3. **Create a KIE.AI account** and top up **$5** (used to render videos in class)
4. **Create a GitHub account** (sign-up only — we'll create the repo in class)
5. **Create a Vercel account** (sign-up only — we'll deploy together in class)

Plus an optional warmup task at the end where you generate one Seedance prompt before class. Strongly recommended — attendees who do the warmup save 30+ minutes in the session.

---

## Before you arrive — think about ONE thing

Spend 10 minutes imagining the project you'd like to build during the session:

- A **subject** — a product, brand, fictional firm, anything
- A **6–10 second looping video** that shows it cinematically
- A **single-page website** built around that loop

Don't write anything down. Just have the vision in your head. The skill turns vision into prompt; the prompt turns into video; the video becomes the hero of a website you'll deploy live before the session ends.

---

## What you'll do in the session

1. You describe your subject and website concept to Claude Code
2. The Seedance skill writes a cinema-grade video prompt
3. You paste it into KIE.AI to render the video
4. The Frontend Design skill builds the website around it
5. Push to GitHub, deploy to Vercel
6. You leave with a live URL
